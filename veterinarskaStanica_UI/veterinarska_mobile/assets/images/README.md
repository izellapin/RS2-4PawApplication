# Pet Images

Ovdje možete dodati slike ljubimaca za galeriju na dashboard-u.

## Preporučene slike:
- `rex.jpg` - slika psa Rex
- `luna.jpg` - slika mačke Luna  
- `max.jpg` - slika psa Max
- `bella.jpg` - slika mačke Bella

## Format:
- Format: JPG ili PNG
- Veličina: 300x200px ili slično
- Naziv: koristite imena kao što su navedena gore

## Kako dodati slike:
1. Dodajte slike u ovaj folder
2. Ažurirajte `pubspec.yaml` da uključi assets
3. Ažurirajte dashboard kod da koristi stvarne slike umesto placeholder-a
